Bagaimana Simulasi Ini Bekerja?

Simulasi ini menggunakan model matematika **Geometric Brownian Motion (GBM)**, standar industri untuk memodelkan fluktuasi harga instrumen keuangan. Harga saham esok hari dihitung menggunakan persamaan diferensial stokastik berikut:

S_t = S_{t-1} * exp( (μ - 0.5 * σ²) + σ * Z )
  • $S_t$ & $S_{t-1}$: Harga saham pada hari $t$ dan hari sebelumnya.
  • $exp$: Fungsi eksponensial alamiah ($e^x$).
  • $\mu$ (Drift Harian): Rata-rata tingkat keuntungan harian berdasarkan data historis saham. Merepresentasikan arah tren jangka panjang.
  • $\sigma$ (Volatilitas Harian): Standar deviasi dari keuntungan harian. Menggambarkan intensitas risiko atau kejutan fluktuasi harga saham.
  • $Z$: Nilai acak acak dari distribusi normal standar (Mean = 0, Std Dev = 1), dihasilkan menggunakan algoritma Box-Muller Transform di setiap langkah simulasi. Komponen ini merepresentasikan "berita mengejutkan" (market shocks) yang tidak terduga setiap harinya.
